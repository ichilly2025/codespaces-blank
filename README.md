# 温度波形图模拟器

一个用于显示温度波形图的前端项目，使用模拟数据生成温度变化曲线，并通过红、黄、绿三种颜色表示不同的温度警告级别。

## 功能特性

- **实时温度模拟**：生成随时间变化的模拟温度数据
- **可视化波形图**：使用Canvas绘制清晰直观的温度变化曲线
- **多级警告系统**：
  - 绿色：正常温度范围（0-30°C）
  - 黄色：警告温度范围（30-40°C）
  - 红色：危险温度范围（40°C以上）
- **交互控制**：
  - 开始/暂停数据模拟
  - 调整模拟速度
  - 重置图表数据
- **响应式设计**：适配不同屏幕尺寸

## 使用方法

### 1. 快速开始

1. 下载项目文件到本地目录
2. 使用浏览器直接打开 `index.html` 文件
3. 点击"开始模拟"按钮启动温度数据模拟

### 2. 控制选项

- **开始/暂停模拟**：控制温度数据的生成和图表更新
- **模拟速度**：调整滑块控制数据更新频率（1x-10x）
- **重置图表**：清除当前图表数据并重新开始

### 3. 温度警告级别

- **绿色区域 (0-30°C)**：正常温度，安全范围
- **黄色区域 (30-40°C)**：警告温度，需要注意
- **红色区域 (40°C以上)**：危险温度，需要立即处理

## 技术实现

### 数据模拟
- 使用正弦波和随机噪声生成逼真的温度变化曲线
- 模拟真实环境中的温度波动模式
- 可配置的温度范围和变化幅度

### 图表绘制
- 使用HTML5 Canvas进行高性能图形渲染
- 实时更新温度曲线
- 平滑的动画过渡效果
- 清晰的坐标轴和刻度标记

### 颜色逻辑
- 基于温度值动态计算曲线颜色
- 平滑的颜色过渡效果
- 直观的视觉反馈系统

## 自定义配置

如需修改温度范围或警告阈值，可在 `app.js` 文件中调整以下配置：

```javascript
// 温度范围配置
const TEMP_CONFIG = {
  minTemp: 0,      // 最低温度
  maxTemp: 50,     // 最高温度
  normalMax: 30,   // 正常温度上限
  warningMax: 40   // 警告温度上限
};
```

## 故障排除

### 常见问题

1. **图表不显示**
   - 检查浏览器是否支持Canvas
   - 确保所有文件在同一目录下
   - 查看浏览器控制台是否有错误信息

2. **按钮无响应**
   - 刷新页面重新加载
   - 检查JavaScript是否被浏览器阻止

3. **性能问题**
   - 降低模拟速度
   - 减少显示的数据点数量

如有问题或建议，请检查代码注释或参考相关HTML5 Canvas文档。
