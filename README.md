# 温度波形图模拟器

一个用于显示温度波形图的前端项目，使用模拟数据生成温度变化曲线，并通过红、黄、绿三种颜色表示不同的温度警告级别。

## 功能特性

- **实时温度模拟**：生成随时间变化的模拟温度数据，温度波动更真实
- **平滑波形曲线**：使用Canvas和贝塞尔曲线绘制平滑的温度变化曲线
- **实时滚动显示**：波形图从右向左滚动，显示最近50个数据点
- **多级警告系统**：
  - 绿色：正常温度范围（< 30°C）
  - 黄色：警告温度范围（30-35°C）
  - 红色：危险温度范围（> 35°C）
- **交互控制**：
  - 开始模拟：启动温度数据生成
  - 暂停模拟：暂停数据更新
  - 重置数据：清除所有数据重新开始
- **响应式设计**：适配不同屏幕尺寸

## 快速启动

### 方法一：直接打开（推荐）

1. 克隆或下载项目到本地：
```bash
git clone git@github.com:ichilly2025/temp-simulator.git
cd temp-simulator
```

2. 直接在浏览器中打开 `index.html` 文件即可

### 方法二：使用本地服务器

如果需要避免浏览器缓存或测试更复杂的功能，可以使用本地HTTP服务器：

**使用Python（推荐）：**
```bash
# Python 3.x
python3 -m http.server 8080

# 或 Python 2.x
python -m SimpleHTTPServer 8080
```

**使用Node.js：**
```bash
# 安装http-server（全局）
npm install -g http-server

# 启动服务器
http-server -p 8080
```

**使用PHP：**
```bash
php -S localhost:8080
```

3. 在浏览器中访问 `http://localhost:8080`

## 使用说明

### 控制面板

- **开始模拟**：点击后开始生成温度数据，波形图实时更新
- **暂停模拟**：暂停数据生成，波形图停止更新
- **重置数据**：清除当前所有数据，重新生成初始温度曲线

### 温度显示

- **波形图区域**：显示最近50个温度数据点的平滑曲线
- **实时温度标签**：波形图右侧显示当前最新温度值
- **当前温度卡片**：页面下方显示当前温度的数字和状态
- **状态指示器**：显示当前温度的警告级别

### 温度警告级别

- **绿色（安全）**：温度 < 30°C，正常范围
- **黄色（警告）**：温度 30-35°C，需要注意
- **红色（危险）**：温度 > 35°C，需要立即处理

## 技术实现

### 数据模拟
- 使用随机波动和趋势性变化生成逼真的温度数据
- 温度波动范围可配置（当前为±8°C）
- 每2秒更新一次数据
- 自动滚动显示最近50个数据点

### 图表绘制
- 使用HTML5 Canvas进行高性能图形渲染
- 采用二次贝塞尔曲线实现平滑的温度曲线
- 实时从右向左滚动显示
- 动态颜色标识（根据温度等级变化）
- 温度标签显示在图表右侧外部，避免遮挡曲线

### 警告系统
- 实时监测温度变化
- 根据预设阈值自动变更警告级别
- 视觉反馈：颜色变化 + 状态指示器

## 技术栈

- **纯原生前端**：HTML5 + CSS3 + JavaScript（ES6+）
- **无需依赖**：不依赖任何外部库或框架
- **Canvas API**：用于图表绘制
- **响应式设计**：适配各种屏幕尺寸

## 开发说明

### 修改温度阈值

在 `app.js` 中修改 `CONFIG.TEMPERATURE` 配置：

```javascript
TEMPERATURE: {
    MIN: -10,          // 最小温度
    MAX: 50,           // 最大温度
    SAFE_MAX: 30,      // 安全温度上限（绿色/黄色分界线）
    WARNING_MAX: 35,   // 警告温度上限（黄色/红色分界线）
    DANGER_MIN: 35     // 危险温度下限
}
```

### 修改更新频率

在 `app.js` 中修改 `CONFIG.SIMULATION` 配置：

```javascript
SIMULATION: {
    DATA_POINTS: 50,       // 显示的数据点数量
    UPDATE_INTERVAL: 2000, // 更新间隔（毫秒）
    TEMP_VARIATION: 8      // 温度波动范围
}
```

## 故障排除

### 常见问题

1. **图表不显示**
   - 检查浏览器是否支持Canvas
   - 确保所有文件在同一目录下
   - 查看浏览器控制台是否有错误信息

2. **按钮无响应**
   - 刷新页面重新加载
   - 检查JavaScript是否被浏览器阻止

3. **性能问题**
   - 降低模拟速度
   - 减少显示的数据点数量
